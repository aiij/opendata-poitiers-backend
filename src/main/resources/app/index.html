À propos

Cette API a été créée par la société Serli. Elle permet d'accéder aux données Open Data de Grand Poitiers. La première version de cette API est destinées à l'application iOS Poitiers Vélo.

La seconde version de cette API contient plus de source :

  • Les Stationnement vélos, le fichier data.gouv.fr
  • Les Bornes à verre, le fichier data.gouv.fr
  • Les Places handicapées GIG / GIC, le fichier data.gouv.fr
  • Les Défibrillateurs, le fichier data.gouv.fr

Des sources de données seront ajoutées au fil du temps.

Vous pouvez contribuer à cette API sur GitHub : https://github.com/Serli/opendata-poitiers-backend

Un tableau de bord permettant de visualiser ces données est disponible sur ce site, pour une expérience plus complète rendez-vous sur Kibana Open Data Poitiers : http://open-data-poitiers-kibana.herokuapp.com/
